# less-plugin-no-comments
Retira todos os comentários da saída CSS do LESS.

## Como instalar

```
npm install -g less-plugin-no-comments
```

## Como utilizar (linha de comando)

```
lessc arquivo.less arquivo.css --no-comments
```
